# git_help

Команда git status показывает список файлов, у которых прошли изменения:  

    git status
    
Добавить сделанные изменения:  

    git add . -p
    
Сделать коммит  

    git commit -m "<Comment>":
    
Сделать пустой коммит:  

    git commit --allow-empty -m "Пустой коммит"
